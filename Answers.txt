1)Git Nedir?
Git,bir sürüm kontol sistemidir ve projelerin daha düzenli bir şekilde yönetilmesinde yardımcı olur,ekip içinde işbirliğini kolaylaştırır.
****************************************************
2)"git pull" ile "git fetch" komutlarının farkı nedir?
Fetch uzak depodan güncellemeleri çeker bilgisini verir git pull ise hem güncelleme alır hem de yerel kopyayı günceller.
****************************************************
3)Eğer takım arkadaşımız "kodlarımı gönderdim, benim geliştirmemin üzerine devam et" derse ve gönderdiği kodları "git pull" ile lokalimize alamıyorsak nerelerde hata yapılmış olabilir?
Kodda aynı dosyanın aynı satırında farklı değişiklikler yapılmış olabilir git hangisini seçmemiz gerektiğini sorar.Git pust yapılmamış olabilir yada git fetch yapılmadan git pullanmışta olabilir.
